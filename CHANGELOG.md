# Changelog

All notable changes to `wethod-cli` will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- `wethod uninstall` command to remove the binary and, optionally, stored data (#6)
- One-line install scripts for macOS, Linux and Windows (#5)

### Changed

- Rewrote the README with a full installation and usage guide (#4)
- Corrected the documented minimum PHP version to 8.4

## [0.0.4] - 2026-06-15

### Changed

- Build the release PHAR on PHP 8.4 (#3)

## [0.0.3] - 2026-06-15

### Added

- Build and publish the standalone PHAR on every GitHub release (#2)

## [0.0.2] - 2026-06-15

- Maintenance release.

## [0.0.1] - 2026-06-15

### Added

- Initial release: a Wethod CLI generated from the OpenAPI spec, with
  authentication, multi-company credentials and the bundled agent skill (#1)
