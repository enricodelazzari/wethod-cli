# Security Policy

## Supported versions

Only the latest release of the Wethod CLI receives security updates. Please make
sure you are running the most recent version (`wethod self-update`) before
reporting an issue.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a vulnerability

**Please do not report security vulnerabilities through public GitHub issues,
discussions, or pull requests.**

Instead, report them privately through GitHub's security advisory feature:

➡️ **[Report a vulnerability](https://github.com/enricodelazzari/wethod-cli/security/advisories/new)**

Please include as much of the following as you can:

- The type of issue and the affected component or command
- Steps to reproduce, including any required configuration
- The impact of the issue and how an attacker might exploit it

You will receive a response acknowledging the report, and we will keep you
informed as the issue is investigated and resolved. Please give us a reasonable
amount of time to address the issue before any public disclosure.

Thank you for helping keep the Wethod CLI and its users safe.
